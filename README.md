## Titanic Exploratory Data Analysis (EDA)
Project Overview

This project performs Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic dataset from Kaggle. The objective is to understand the dataset, clean missing values, visualize important features, and identify patterns and relationships that influence passenger survival.



## Objective

- Perform data cleaning on the dataset.
- Explore numerical and categorical variables.
- Identify missing values and duplicates.
- Analyze relationships between variables.
- Visualize trends using charts and graphs.



## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab



##  Dataset

Dataset: Titanic - Machine Learning from Disaster (Kaggle)

Files used:
- train.csv


## Data Cleaning

- Checked dataset structure
- Identified missing values
- Filled missing values in Age using Median
- Filled missing values in Embarked using Mode
- Removed the Cabin column due to excessive missing values
- Checked and removed duplicate records

---

## Exploratory Data Analysis

The following analyses were performed:

- Survival Distribution
- Gender Distribution
- Passenger Class Distribution
- Age Distribution
- Fare Distribution
- Survival by Gender
- Survival by Passenger Class
- Fare by Passenger Class
- Correlation Heatmap

---

##  Key Findings

- Most passengers did not survive.
- Male passengers were more than female passengers.
- Female passengers had a higher survival rate.
- Most passengers belonged to Third Class.
- Most passengers were between 20 and 40 years old.
- First-class passengers paid higher fares and had better survival rates.
- Passenger class, gender, and fare influenced survival.

---

## Conclusion

The Titanic dataset was successfully cleaned and analyzed using Exploratory Data Analysis (EDA). The analysis revealed meaningful insights into passenger demographics and survival patterns, demonstrating the importance of data cleaning and visualization before predictive modeling.
